# Power BI Theme Usage

## Theme File

Use:

- `bi/powerbi/workforce_intelligence_theme.json`

## Import Steps

1. Open the report in Power BI Desktop.
2. Go to `View`.
3. Select `Browse for themes`.
4. Import `workforce_intelligence_theme.json`.

## What This Theme Changes

- light executive background
- neutral white visual cards
- blue-led analytical palette
- amber for caution
- red for high-risk storytelling
- cleaner typography for cards, slicers, and charts

## Recommended Manual Follow-Up

The theme improves the visual baseline, but these still need manual layout adjustment:

- align KPI cards to the same width and height
- keep slicers in a compact horizontal row
- apply red, amber, and blue intentionally to risk-band visuals
- reduce table dominance on the Attrition Risk and Labor Market pages
